What I have done today

  1. Ran the training script with a little change.
    • Most errors I met before caused by version corresponding among torch, torchvision, cuda, cudnn. Especially, remove _pytorch.select package in conda to use pytorch with gpu. Recreate environment of ‘rsfin’ followed by: Runs and errors occurred in line 23 torchvision.transforms.RandomRotation(20, resample=Image.BILINEAR): This caused by torchvision version changing. See help(torchvision.transforms.RandomRotation) in python and report: Interpolation and resample are two kinds of method in image processing. The difference between them is that interpolation is one way to realize resampling for separate data, see. And the difference between NEAREST and BILINEAR is that BILINEAR is more delicate, see. So change the code to torchvision.transforms.RandomRotation(20, interpolation=Image.BILINEAR) and all will be fine.
